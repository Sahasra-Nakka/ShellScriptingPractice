# Shell Scripting Practice
## Committed Files
### hellouser.sh
This program uses read and echo commands along with a -p flag.
read command takes an input from the user and echo command prints a statement in the command line.
-p shows the prompt message.

### oddoreven.sh
The commands used in this program are read and echo.
The program consists of an if-then-else block which ends with fi marking the end of the conditional block.
-eq comparision operator is used in the if condition.

### positiveornegative.sh
if-elif-else block is used in this program.
For conditional checking -gt(greater than) and -lt(less than) comparision operators are used.

### printnumbers.sh
A for loop is used to iterate through a range given.

### filecheck.sh
-f flag is used to check if the file is present or not.
